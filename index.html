C++/cpp入门教程

2026-07-10

本教程默认读者已经装好编译器，编译器（VS2022 MSVC）配置教程择期发布。

#include <iostream> int main(){ std::cout<<"Hello World!"<<endl; return 0; }

让我们从第一行开始。

第一行：#include <iostream>

include，意为导入，其意义便是导入头文件。
头文件是大部分C++程序运行时不可或缺的东西，接下来，例如vector，algorithm，map，queue等头文件会时常被我们提起。
请注意，许多教程都会提到一个叫做bit/stdc++.h的东西，这个东西常被称为万能头文件。但在我们讲解中使用的MSVC编译器中没有该头文件，强制引用会报错。

第二行：int main()

这是C++程序运行的主函数，它不可以改名，不可以不写，并且在初学阶段我强烈不推荐任何人使用void main，signed main之类的写法。

第三行 std::cout<<”Hello World!”<<endl;

std::是标准命名空间前缀，今后想参加大型程序开发的各位，一定不要偷懒去写什么using namespace std;，没有std:: 很容易报各种莫名其妙的错误。
cout是输出流函数，用于输出（想用C风格输出欢迎使用printf）。

表示输出流，在今后的字符串操作中我们会详细讲到它
“Hello World!” 表示一个内容为 Hello World! 的字符串，同样，对于此类字符串操作，我们同样在后面会讲到它。
注意，绝对不要写成’’单引号

第五行 return 0;

返回值，默认返回0表示程序运行成功（并不代表没有bug）

第一期到此结束
